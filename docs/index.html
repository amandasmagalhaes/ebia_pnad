1 Data Source and Identification of Food Insecurity Variables

1.1 PNAD Continuous

The Continuous National Household Sample Survey (PNAD Continuous) is a nationwide household survey conducted by the Brazilian Institute of Geography and Statistics (IBGE).

More information about the survey is available on the PNAD Continuous webpage.

1.1.1 Accessing the Documentation

To identify food insecurity variables, we accessed the PNAD Continuous Microdata page.

Within the Microdados da Divulgação Anual and then Concentrados em determinado trimestre section, the following documents were consulted:

  • LEIA-ME.pdf
  • PNADC_Pesquisas_Suplementares_Anuais_20260508.pdf

To identify the variables included in the Food Security module, we then accessed:

Trimestre_4 → Documentação

and consulted:

  • dicionario_PNADC_microdados_trimestre4_20251010.xls

1.1.2 Food Security Module

According to the supplementary survey documentation, the Food Security module was collected only in the fourth quarter of 2023 and 2024.

The module corresponds to the Brazilian Food Insecurity Scale (EBIA) and includes 14 questions related to household experiences of food insecurity during the previous three months.

The questionnaire variables are:

  • S17001–S17004: questions asked in all households.
  • S17005–S17008: additional questions for households with adults (18+ years).
  • S17009–S17014: additional questions for households with children or adolescents (0–17 years).

Responses are coded as:

  • 1 = Yes
  • 2 = No

1.1.3 Food Insecurity Variable

In addition to the individual questionnaire items, IBGE provides a derived variable containing the final food security classification:

Variable Description
SD17001 Household food security status

Categories:

Code Classification
1 Food security
2 Mild food insecurity
3 Moderate food insecurity
4 Severe food insecurity

For most analyses, SD17001 can be used directly, as it contains the official EBIA classification produced by IBGE.

1.1.4 Using the PNADcIBGE R Package

The PNADcIBGE package provides direct access to official PNAD Continuous microdata from IBGE.

The package is available on CRAN: PNADcIBGE.

The package can be used to download and import PNAD Continuous datasets directly into R.

Capital Status Distribution 
# Plot the distribution of observations located in state capitals versus all other municipalities
bind_rows(
  pnad_2023 %>% mutate(Year = "2023"),
  pnad_2024 %>% mutate(Year = "2024")
) %>%
  mutate(
    CapitalStatus = if_else(
      is.na(Capital),
      "Other municipalities",
      "State capitals"
    )
  ) %>%
  count(Year, CapitalStatus) %>%
  group_by(Year) %>%
  mutate(
    Percent = 100 * n / sum(n),
    Label = paste0(
      scales::comma(n),
      " (",
      round(Percent, 1),
      "%)"
    )
  ) %>%
  ungroup() %>%
  ggplot(
    aes(
      x = CapitalStatus,
      y = n
    )
  ) +
  geom_col(
    fill = "steelblue"
  ) +
  geom_text(
    aes(label = Label),
    vjust = -0.3,
    size = 4
  ) +
  facet_wrap(~Year) +
  scale_y_continuous(
    labels = scales::comma,
    expand = expansion(mult = c(0, 0.1))
  ) +
  labs(
    title = "Distribution of Observations by Capital Status \n",
    x = NULL,
    y = "Number of observations \n"
  ) +
  theme_minimal() +
  theme(
    plot.title = element_text(
      face = "bold",
      hjust = 0.5
    ),
    strip.text = element_text(
      face = "bold"
    )
  )

1.2 PNAD

Before the implementation of PNAD Continuous, IBGE conducted the National Household Sample Survey (PNAD), an annual household survey that collected demographic, social, and economic information on the Brazilian population.

More information about the survey is available on the PNAD webpage.

The annual PNAD was discontinued in 2016 and replaced by PNAD Continuous. However, historical PNAD microdata remain available and are frequently used for long-term analyses and historical comparisons.

1.2.1 Accessing PNAD Microdata

PNAD microdata can be accessed through the PNAD Microdata page.

Each annual folder includes a ZIP file containing the survey data and the corresponding documentation required to identify variables and reproduce analyses.

Researchers interested in historical food insecurity estimates should consult the documentation for the 2004, 2009, and 2013 survey rounds, which include the EBIA module.

1.3 POF

Food insecurity information is also available in the Brazilian Household Budget Survey (POF), conducted by IBGE.

More information about the survey is available on the POF webpage.

According to the IBGE report on food insecurity, EBIA information is available in the POF 2017–2018. The most recent POF (2024–2025) has been conducted, but its microdata were not yet available at the time this document was prepared.

1.3.1 Accessing POF Microdata

POF microdata can be accessed through the POF Microdata page.

The most relevant section for researchers is Dados, Documentação e Leia-me.
